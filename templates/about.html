Bodegas to Supermarket Ratio By Neighborhood in New York City

In order to find something that provides insight on our communities, I wanted to do something about food. A big issue surrounding communities in New York is the access to fresh produce for good diets; many communities don't have that many supermarkets, so they rely on bodegas and delis for food.

Some feedback I got for this project is that I have to make sure I include both delis and bodegas, as they provide the same resources as each other. Some takeaways I had from the feedback is that I should definitely organize my data using a map of New York City, and make sure I am displaying useful information, as well as making it organized.

Here is my data pipeline:

  • The data set used for this project was downloaded from the Department of Environment & Health's website: https://a816-dohbesp.nyc.gov/IndicatorPublic/VisualizationData.aspx?id=2389,719b87,127,Disparities,Ratio%20of%20Bodegas%20to%20Supermarkets,years=2016,dataLink=Neighborhood%20Poverty
  • Google Sheets was used to eliminate columns that were not needed for this project, so it strictly focuses on the bodega and supermarket ratio
  • I formatted the data into a dictionary so I can see the storage by year and by county
  • I used a similar python program to the file from LifeExpectancy to read the data and turn it into a .json file
  • The final step was to interpret the data and display it
